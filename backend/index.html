SIPEKAN API Documentation

API digunakan untuk project capstone SIPEKAN - Sistem Pengelola Keseharian Anak oleh Tim C22-082.

Endpoint

https://api.sipekan.my.id

Auth

Register

  • URL
    • /register
  • Method
    • POST
  • Request Body
    • name as string
    • email as string, must be unique
    • password as string
    • confirmPassword as string
  • Response 
    {
  "status": "Berhasil",
  "pesan": "User Berhasil Didaftarkan"
}

Login

  • URL
    • /login
  • Method
    • POST
  • Request Body
    • email as string
    • password as string
  • Response 
    {
    "status": "Berhasil",
    "pesan": "Anda Berhasil Login",
    "data": {
        "user_id": "ecbaa95d-84fe-4809-a9e1-b8e05126ab7b",
        "name": "ayah",
        "email": "ayah@gmail.com",
        "role": "admin"
    }
}

User

Add New User

  • URL
    • /user
  • Method
    • POST
  • Request Body
    • name as string
    • email as string, must be unique
    • password as string
    • confirmPassword as string
    • role as admin or user
    • point as integer
  • Response 
    {
    "status": "Berhasil",
    "pesan": "User Berhasil Ditambahkan"
}

  • Headers
